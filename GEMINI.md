# GAMAZ Carpintería - AI Development Guidelines

## Project Overview

GAMAZ Carpintería is a family-owned carpentry business in Mexico.

The goal of this website is to generate customer inquiries and WhatsApp quotations.

The website should emphasize:

- Trust
- Craftsmanship
- Professionalism
- Family values
- High-quality work

## Technical Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel Deployment

## Development Principles

- Prioritize simplicity over complexity.
- Use Server Components whenever possible.
- Create reusable components.
- Keep code production-ready.
- Maintain accessibility best practices.
- Optimize for mobile devices first.

## Business Goals

The website is NOT an online catalog.

The website IS a lead-generation tool.

Every design decision should answer:

"Will this increase trust and encourage users to request a quote?"

## Design Direction

Style:

- Premium
- Minimalist
- Warm
- Modern craftsmanship

Reference:

Apple × Artisan Woodworking

Colors:

- Walnut brown
- Warm white
- Olive green accents

## Pages

Current priority:

- Home page

Future pages:

- Projects
- Contact
- About Us

## CTA

The primary CTA throughout the site is:

"Solicitar cotización por WhatsApp"

## Coding Standards

- Avoid unnecessary dependencies.
- Write clean TypeScript.
- Use descriptive component names.
- Document non-obvious logic.
- Favor maintainability over cleverness.
